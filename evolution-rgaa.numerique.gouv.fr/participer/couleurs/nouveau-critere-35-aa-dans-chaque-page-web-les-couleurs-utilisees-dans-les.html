  • Proposition de OLIVIER NOURRY | 14/06/2019 - 09:51

    Autoriser la technique consistant à satisfaire le critère via une version alternative

    Signalez un abus

    La technique G174 (https://www.w3.org/WAI/WCAG21/Techniques/general/G174), qui consiste à permettre à l'utilisateur de sélectionner une feuille de style alternative, est considérée comme suffisante pour le critère 1.4.11. Pourtant elle n'est pas du tout prise en compte pour ce nouveau critère.
    Proposition: compléter les tests pour rendre conformes les contenus qui passent le critère de couleur si une feuille de style alternative est appliquée, dans les mêmes conditions que les critères 3.3 et 3.4.
